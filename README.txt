Wyre WooCommerce  plugin allows you to accept payment from local and international customers on your store.


== Description ==
Wyre is a unique online payment processor whose vision is to offer buyers and sellers a secure and easy-to-use means of transacting business online.

Wyre allows site owners to receive payment for their goods and services on their website without any setup fee.

Wyre distinguishes itself from other online payment processors with its array of features that are guided by its core values:

* <strong>Security</strong>
* <strong>User Friendliness</strong>
* <strong>Reliability</strong>

To signup on Wyre, visit  [here](https://Wyre.com/3445-0056682)


= Note =

Before you start accepting payment on your site, login to your [Wyre Account](https://Wyre.com/login) go to <strong>Account Settings</strong> then <strong>API Settings</strong>, then turn on <strong>Enable Notification API</strong>. Then click on <strong>Save Changes</strong> for the changes to be effected. <br /><strong>It is very important that you do this before you start accepting payment on your site.</strong>


= Plugin Features =

* Demo mode
* Custom Plugin name and description
* Redirect or Inline payment methods
* Inline payment custom message
* Sending customer billing information during payment
 
 
== Installation ==

= Installation =
* 	Login to your WordPress Admin area
* 	Go to "Plugins > Add New" from the left hand menu
* 	In the search box type "Wyre WooCommerce"
*	From the search result you will see "Wyre WooCommerce" click on "Install" to install the plugin
*	click "Activate" to activate it

= Note: =
If this is the first time of installing a WordPress Plugin, you may need to enter the FTP login credential information. This information is available through your web server host.


= Manual Installation =
1. Download the plugin zip file
2. Create a folder "woo-Wyre" in wp-content/plugins and extract all contents from the zip into it *or* upload the zip via the plugin section on the admin page.
3. Click installed plugins in plugin tab
4. Click activate to enable this on woocommerce  
5. Click settings under "Wyre WooCommerce" to configure the plugin

= Configure the plugin =

* Wyre Merchant ID: This can be found in your Wyre account

* Demo mode: Enable this for payment TESTING PURPOSE ONLY

* Payment Method: Inline method loads the checkout page directly on your website without redirection

* Inline Load Messsage: You can add a loading message for your customers while the checkout is being loaded.

* Customer Infomation: Send customer billing address and contact information. This helps in ensuring a successful transaction.
